# Course Selection HiFi

## How to Run (Hi‑Fi Prototype)

1. Open the project folder in **VS Code**.
2. Open `index.html`.
3. Right‑click inside the file and choose **“Open with Live Server”** (VS Code Live Server extension required).

## View as iPhone SE

1. With the page open in your browser, **right‑click → Inspect** 
2. Toggle **Device Toolbar** (phone/tablet icon)
3. Select **iPhone SE** from the device dropdown.
4. Make sure the viewport is aligned to start **from the very top of the screen**.
5. If needed, manually adjust width/height in the responsive view to fine‑tune the layout.
